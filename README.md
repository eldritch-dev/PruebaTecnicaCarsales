# PruebaTecnicaCarsales
Backend for Frontend API.

# Consideraciones previas
1. Visitar https://dotnet.microsoft.com/es-es/download/dotnet/9.0 y descargar e instalar .NET SDK 9.0.308.
2. Visitar https://dotnet.microsoft.com/en-us/download/dotnet/8.0 y descargar e instalar ASP.NET Core Runtime 8.0.22.
3. El proyecto se escribió y revisó por mi parte en Visual Studio Code. Para levantar el proyecto en Visual Studio Code, instalar las siguientes dependencias.
    2.1. .NET Install Tool
    2.2. C# Dev Kit

# Instrucciones para probar el proyecto
1. En terminal git bash o powershell ejecutar "dotnet run".
2. Al terminar de revisar el proyecto, detener servidor con "Ctrl+c" en la terminal.

# Endpoints base
1. Para que el frontend pueda consumir recursos: http://localhost:5208/api/
2. Para revisar en Swagger UI en el navegador, si es que no se abre automáticamente: http://localhost:5208/swagger
